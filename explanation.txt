### Overview of the Library Design

The proposed library, which I'll call `ValueInvestPy`, is designed as a lightweight, user-friendly Python package tailored for casual retail investors practicing value investing. Value investing emphasizes identifying undervalued stocks through fundamental analysis, focusing on long-term holding rather than short-term trading. This library prioritizes simplicity, accessibility, and integration with free/open data sources to avoid subscription costs.

Key principles guiding the design:
- **Ease of Use**: Minimal setup, intuitive APIs, and clear documentation.
- **Focus on Fundamentals**: Tools for fetching and analyzing company financials, calculating key value metrics (e.g., P/E ratio, intrinsic value via discounted cash flow).
- **Casual Scope**: No advanced quantitative features like high-frequency trading, derivatives pricing, or machine learning models. Instead, emphasize screening, basic portfolio tracking, and visualizations.
- **Dependencies**: Built on popular open-source libraries to keep it free and extensible:
  - `yfinance` for stock data.
  - `pandas` and `numpy` for data manipulation.
  - `matplotlib` and `seaborn` for visualizations.
  - `requests` and `beautifulsoup4` for scraping additional data if needed (e.g., from free APIs like Alpha Vantage or SEC EDGAR filings).
  - No paid APIs to suit casual users.
- **Structure**: Organized into modular subpackages for clarity. The library would be installable via pip (e.g., as a GitHub repo or PyPI package).
- **Limitations**: Relies on public data, which may have delays or inaccuracies. Users should verify with official sources.

The library would include example Jupyter notebooks for tutorials, and error handling for API rate limits or data unavailability.

### Core Modules and Features

The library is divided into the following modules, each with key functions. I'll describe the purpose, main functions, and example usage for each.

#### 1. **Data Fetch Module (`valueinvestpy.data_fetch`)**

This module handles retrieving historical and real-time data from free sources. Essential for value investors to access balance sheets, income statements, and stock prices without manual downloads.

- **Key Functions**:
  - `fetch_stock_data(ticker, period='5y')`: Retrieves historical prices, dividends, and splits using yfinance.
  - `fetch_financials(ticker)`: Pulls income statement, balance sheet, and cash flow data.
  - `fetch_economic_indicators(indicator='GDP')`: Fetches macro data (e.g., GDP, inflation) from public APIs like FRED (Federal Reserve Economic Data).
  - `fetch_peer_data(sector='Technology')`: Gets data for comparable companies in a sector.

- **Example Usage**:
  ```python
  from valueinvestpy.data_fetch import fetch_stock_data, fetch_financials

  data = fetch_stock_data('AAPL', period='10y')
  financials = fetch_financials('AAPL')
  print(financials['income_statement'].head())
  ```

#### 2. **Fundamental Analysis Module (`valueinvestpy.analysis`)**

Core to value investing: Computes ratios and estimates intrinsic value. Includes Benjamin Graham-inspired formulas (e.g., net current asset value) and basic DCF (Discounted Cash Flow) models.

- **Key Functions**:
  - `calculate_ratios(financials)`: Computes key metrics like P/E, P/B, ROE, debt-to-equity, current ratio, and earnings yield.
  - `estimate_intrinsic_value(ticker, growth_rate=0.05, discount_rate=0.1)`: Simple DCF model using free cash flow projections.
  - `graham_number(financials)`: Calculates Graham's intrinsic value formula (sqrt(22.5 * EPS * Book Value)).
  - `margin_of_safety(current_price, intrinsic_value)`: Returns the discount percentage (e.g., buy if >50% undervalued).
  - `quality_score(financials)`: A composite score based on profitability, growth, and stability (e.g., Piotroski F-Score variant).

- **Example Usage**:
  ```python
  from valueinvestpy.analysis import calculate_ratios, estimate_intrinsic_value

  ratios = calculate_ratios(financials)
  intrinsic = estimate_intrinsic_value('AAPL', growth_rate=0.03)
  print(f"P/E Ratio: {ratios['pe']}, Intrinsic Value: {intrinsic}")
  ```

#### 3. **Stock Screening Module (`valueinvestpy.screening`)**

Allows users to filter stocks based on value criteria, scanning a universe (e.g., S&P 500) for undervalued opportunities.

- **Key Functions**:
  - `screen_stocks(universe='SP500', criteria={'pe_max': 15, 'pb_max': 1.5, 'roe_min': 0.15})`: Filters based on user-defined thresholds for ratios.
  - `value_screen(growth_min=0.05, debt_max=0.5)`: Pre-built screen for classic value stocks (low P/E, strong balance sheet).
  - `sector_screen(sector='Consumer Goods')`: Screens within a specific industry.
  - `export_screen(results, format='csv')`: Saves results to file.

- **Example Usage**:
  ```python
  from valueinvestpy.screening import screen_stocks

  results = screen_stocks(criteria={'pe_max': 20, 'div_yield_min': 0.03})
  print(results)  # Pandas DataFrame with tickers and metrics
  ```

#### 4. **Portfolio Management Module (`valueinvestpy.portfolio`)**

Basic tools for tracking a value-oriented portfolio, including diversification checks and performance metrics.

- **Key Functions**:
  - `build_portfolio(tickers, weights=None)`: Creates a portfolio object with current values and historical returns.
  - `calculate_performance(portfolio, benchmark='SPY')`: Computes total return, CAGR, and alpha vs. benchmark.
  - `diversification_check(portfolio)`: Analyzes sector allocation and concentration risk.
  - `rebalance_suggestions(portfolio, target_weights)`: Suggests buys/sells to maintain allocation.
  - `risk_metrics(portfolio)`: Calculates volatility, beta, and Sharpe ratio (using risk-free rate from data_fetch).

- **Example Usage**:
  ```python
  from valueinvestpy.portfolio import build_portfolio, calculate_performance

  pf = build_portfolio(['AAPL', 'KO', 'BRK-B'])
  perf = calculate_performance(pf)
  print(f"CAGR: {perf['cagr']}%")
  ```

#### 5. **Visualization Module (`valueinvestpy.visuals`)**

Graphical tools to visualize data, aiding in quick assessments of value.

- **Key Functions**:
  - `plot_price_history(ticker, with_dividends=True)`: Line chart of prices and dividends.
  - `plot_ratios(financials, ratios=['pe', 'pb'])`: Bar or line charts for historical ratios.
  - `plot_portfolio_allocation(portfolio)`: Pie chart for sector/asset breakdown.
  - `compare_stocks(tickers, metric='roe')`: Side-by-side comparison plots.
  - `save_plot(fig, filename='chart.png')`: Exports visuals.

- **Example Usage**:
  ```python
  from valueinvestpy.visuals import plot_price_history

  plot_price_history('AAPL').show()
  ```

#### 6. **Backtesting Module (`valueinvestpy.backtest`)**

Simple historical testing of value strategies, e.g., buying low P/E stocks annually.

- **Key Functions**:
  - `run_backtest(strategy='low_pe', universe='SP500', start_date='2010-01-01', rebalance_freq='yearly')`: Simulates returns.
  - `define_custom_strategy(criteria)`: Allows user-defined rules (e.g., Graham criteria).
  - `performance_summary(results)`: Returns metrics like total return, max drawdown.

- **Example Usage**:
  ```python
  from valueinvestpy.backtest import run_backtest

  results = run_backtest(strategy='value_blend')
  print(results['summary'])
  ```

#### 7. **Utilities Module (`valueinvestpy.utils`)**

Helper functions for common tasks.

- **Key Functions**:
  - `currency_convert(amount, from_cur='USD', to_cur='EUR')`: Basic conversion using fetched rates.
  - `data_cache(ticker)`: Caches fetched data to avoid repeated API calls.
  - `error_handler(func)`: Decorator for graceful handling of API errors.
  - `export_report(portfolio, filename='report.pdf')`: Generates a PDF summary using a library like `reportlab`.

### Implementation Considerations

- **Data Sources**: Primarily yfinance for speed and reliability. Fallback to Alpha Vantage (free tier) or web scraping for EDGAR filings if needed.
- **Error Handling and Logging**: Built-in logging to track API issues; user-friendly messages like "Data unavailable—try again later."
- **Testing**: Unit tests for each function using pytest, with mock data to simulate API responses.
- **Documentation**: Docstrings for all functions, plus a README with setup instructions and examples.
- **Extensibility**: Users can subclass modules for custom strategies.
- **Potential Enhancements**: If users want more, integrate optional ML via scikit-learn for predictive scoring, but keep it out of core for casual focus.

This design provides a solid foundation for a retail value investor to research, screen, and manage investments without overwhelming complexity.
